# Лабораторная №4
Изучены интерфейсы и абстрактные классы
## Реализовано
Созданы 2 интерфейса - Delivery_avaliable - наличие метода для подсчета доставки и Printable - наличие метода, который выдает все сведения об объекте

Класс Customer наследует метод Printable, его дочерние классы - метод Delivery_avaliable
Доставка в разных классах считается по-разному, а сведения об объекте предоставляются соответственно аттрибутам объекта определенного класса
