# Вариант 3 Интернет-магазин

## Класс Customer
 атрибут класса 

 `shop_name = "PC Parts Shop"`

 закрытые поля

 `_name`

 `_email`

 `_wallet_balance`

 `_bonus_points` - можно использовать при покупке, за каждую начисление 5% от суммы

state

`_banned` - bool

 properties 

 dunder methods

`__str__` — читаемое описание для пользователя

`__repr__` — для отладки объекта

`__eq__` — сравнение по email 


 business methods

`balance_increase(self, amount)` - пополнение баланса 

`make_purchase(self, price, use_points)` - совершение покупки

state method

`ban(self)` - деактивировать аккаунт (нельзя совершать покупки и пополнять баланс)

`unban(self)` - активировать аккаунт
