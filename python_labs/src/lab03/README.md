# Лабораторная №3
Полиморфизм, наследование, переопределение методов

Преедметная область: интернет-магазин

Базовый класс - Customer

## Реализовано

Дочерние классы:

 CorporateCustomer, аттрибуты warehouse_distance, order_limit, методы make_purchase и str

 HumanCustomer, аттрибуты phone_number, delivery_method , методы make_purchase и str переопределены

## Сценарий 1
```
корпоративный клиент - совершение покупки 

профиль клиента 

Профиль клиента. 
Имя: x5 group 
Email: x5@mail.ru 
Баланс: 400000 $ 
Бонусные баллы: 0 
Лимит на сумму заказа: 1000000 $ 
Расстояние от магазина до склада: 15 км

до покупки 

Ваш баланс 400000, вы можете сделать заказ на сумму 1000000, доставка будет стоить 1687.5

после покупки 

Ваш баланс 363312.5, вы можете сделать заказ на сумму 1000000, доставка будет стоить 1687.5
```

## Сценарий 2
```
обычный клиент - совершение покупки 

Профиль клиента. 
Имя: Steve Jobs 
Email: steve@mail.usa 
Баланс: 35000 $ 
Бонусные баллы: 666 
НОмер телефона: +72222222222
Способ доставки: пункт выдачи
до покупки 

Ваш баланс 35000, ваши баллы 666, доставка бдует стоить 10
после покупки 

Ваш баланс 34820, ваши баллы 646.0, доставка бдует стоить 10
генерация кода выдачи - последние 4 цифры номера

2222
```
